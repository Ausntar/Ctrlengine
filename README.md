# Ctrlengine
###		使用C++编写的高性能游戏引擎
#### Ctrlengine Development version 1.1 [10H2]
               
使用库：
1. SpdLog(https://github.com/gabime/spdlog)
2. Assimp(https://github.com/assimp/assimp)
3. Bass(https://www.un4seen.com/)
4. GLFW(https://www.glfw.org/)
5. nlohmann-json(https://github.com/nlohmann/json)
6. stb-master(https://github.com/nothings/stb)
7. Vulkan(https://www.vulkan.org/)

(在使用之前请正确安装这些库！)
